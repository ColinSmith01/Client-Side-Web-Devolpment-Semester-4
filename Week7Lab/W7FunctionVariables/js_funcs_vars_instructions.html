Exercise 1: Random Number

  • Open the files ex1.html and ex1.js in a text editor.
  • In file em.html use a script element to include the file ex1.js. If you cannot remember how to do it, go back to lab Javascript in HTML.
  • In the Javascript file you will find a function called buttonAction. How many parameters does it have? What does it do?

    It has 0 parameters. It does nothing (for the moment) but is a valid function.

  • Add a call to function buttonAction as the handler code for a click on the button in file ex1.html. If you cannot remember how to do it, go back to lab Javascript Events and Handlers.
  • What happens when you click the button now? Why?

    Nothing happens. Because the called function does nothing.

  • In file ex1.js write a new function. It should:
    • be named getRand1to10
    • have 0 parameters
    • get a random number between 1 and 10, inclusive, and assign it to variable r, with the following code:
      
let r = Math.floor(Math.random() * 10) + 1
      How does this work (if you are interested)?
      • Math.random() returns a random real number between 0 (inclusive) and 1 (exclusive)
      • * 10 makes the number 10 times bigger and the number is now a random real number between 0 (inclusive) and 10 (exclusive)
      • the call to Math.floor rounds the number down to the nearest integer and now we have a random integer between 0 (inclusive) and 9 (inclusive)
      • + 1 moves the range of integers from 0-9 to 1-10 and now we have a random integer between 1 (inclusive) and 10 (inclusive)
    • return the obtained random number
  • Open the file ex1.html in a browser, open the developer tools and reload ex1.html. Now you can test your function by calling it from the console.
  • Once your function getRand1to10, add some code to function buttonAction so that it calls getRand1to10 and writes the random number that is returned into the element with id rand-num.
  • Test that the button click makes the random number appear on the page in em.html (see this video for example of expected behaviour).

Exercise 2: Triple the Random Number

  • Create copies of the files ex1.html and ex1.js and rename them ex2.html and ex2.js.
  • Open ex2.html and ex2.js in a text editor.
  • In file ex2.html change the included Javascript to ex2.js.
  • In file ex2.html add the following elements:
    • a button with text 'Click to triple the random number'
    • a paragraph with id 'tripled'
  • In file ex2.js add a function that reads the random number from the element with id 'rand-num', multiplies it by 3 and writes the result into the element with id 'tripled'.
  • You can read the number from the element with id 'rand-num' and triple it with the code
    
document.getElementById("rand-num").innerHTML * 3
    How does this work?
    • document.getElementById("rand-num") returns the element with id 'rand-num'
    • .innerHTML accesses the property called innerHTML on that element (this property has all the contents of the element in a string)
    • even though innerHTML is a string, if it represents a number and is multiplied by a number, the result is a number (try "4" * 2 in the console and note the difference from Python)
  • Add a call to your new function as the handler for clicks on the new button.
  • Test your solution to make sure it does what it should, by first clicking on the button that creates a random number and then on the one that triples it (see this video for example of expected behaviour).
