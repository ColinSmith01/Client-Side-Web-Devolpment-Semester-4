Javascript and HTML Input Elements Exercises

Exercise 1: Calculator

  • Open the files inp_ex1.html and inp_ex1.js in a text editor.
  • The following html elements are defined in the HTML file (find them):
    • two input fields
    • four buttons: "Add", "Subtract", "Multiply" and "Divide"
    • an empty span element with id answer
  • Your task is to write Javascript code that implements a calculator. When any of the buttons is clicked, the requested operation should be performed on the numbers in the two input fields. The result should be displayed in the element with id answer.
  • Place your code in a file called inp_ex1.js and include it in the HTML file using a script element. The script element should be created as the last element inside the body element.
  • Tips:
    • The click handlers have been set for the buttons, indicating the function names you should use when implementing the operations.
    • Start by writing one of the functions and you will find that the others will be similar.
    • Plan out what the function is going to do before you write it (read the inputs, perform the operation, write the output to the answer element).
    • Think how you would do each of these steps, then implement them.
    • The value entered by the user into an input field is stored in the element object property value (not innerHTML).
    • The value read out of the input field is a string and cannot be used directly in arithmetic operations. You can convert the value to a number by prepending a + to the string expression. For example
      
+document.getElementById("num1").value
